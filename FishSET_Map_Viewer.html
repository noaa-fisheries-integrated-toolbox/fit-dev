FishSET-Map-Viewer (FishSET_Map_Viewer)

Authors

Corinne Bassin

Description

This tool allows the display of various data on a map, built specifically for fishing data. It displays and allows for analysis of fishing hauls using the start and end points of the haul. Data associated to an individaul haul is shown on the map and colored based on the value of a choosen variable associated to that data. The data is also spatially grouped based on the users choice of grid.
