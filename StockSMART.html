Stock Status, Management, Assessments & Resource Trends

Tool

Stock Status, Management, Assessments & Resource Trends (StockSMART)

Type

Web Application

Authors

W. Qiu, K. Blackhart, J. Vieser, and A. Furnish

Description

Stock SMART is NOAA Fisheries' Stock Status, Management, Assessment, and Resource Trends web tool. Stock SMART includes different applications for users to search, view, compare, and download the results of assessments for stocks managed by NOAA Fisheries. Future updates will allow users to access stock status determinations and other related information on fisheries management.
