Tool: AD Model Builder

AD Model Builder (ADMB)

Type:

Model Builder

Description:

This tool allows the display of various data on a map, built specifically for fishing data wherein thir is a location to thte start and end of a fishing haul. Data associated to an individaul haul is shown on the map and colored based on the value of a choosen variable associated to that data. The data is also spatially grouped based on the users choice of grid.
